Open Source Software for Windows

There is some great open source software for the Windows platform. Finding it is sometimes difficult. Here is a list of some of the software and some of the sites - so I don't have to keep searching Google for it.

Sites

Open Source Software in a Windows Environment

The OpenCD Project and GNU Win- contains a rather complete list of links to open source software.

The Software for Starving Students CD enhances the Windows and Mac desktop computing experiences by providing an easy way to install free, high-quality software titles via a user-friendly interface.

Freshmeat lists the Win32 projects by popularity and by rating.

Windows GNU Gaming Zone

Internet

Mozilla - browser, email, chat, html editor.

Firefox - a redesign of the Mozilla browser component.

Thunderbird - a redesign of the Mozilla mail component.

Calendar - a standards-based calendar client.

Filezilla - a fast and reliable FTP client and server with lots of useful features and an intuitive interface.

PuTTY - terminal for SSH and Telnet

Gnucleus and Shareaza - Gnutella P2P Clients

XChat - IRC chatting tool

PSI - a Jabber client for the open instant messaging protocol

Editors, Word Processing, Spreadsheets, Photoeditor

MBEdit - an AEDIT clone, with syntax highlighting and other improvements. Built for multiple platforms. See the original AEDIT User's Manual.

GVIM - an vi clone, vi-improved (with syntax highlighting).

OpenOffice - complete office suite. See my OpenOffice Resource Page.

PDF Creator - tool for creating PDF documents from Postscript documents, which can be created by setting up a postscript printer (albeit a phony printer), and printing to that printer, and having that printer print to a file.

Ghostscript and GSView - tools for viewing/converting postscript or PDF files.

GIMP - GNU Image Manipulation Program for Windows or Linux.

phpedit - an IDE (Integrated Developement Environment) to work with PHP in a Windows enviroment.

Games

BZFlag - a BattleZone-like capture the flag multiplayer game.

Tux Racer - you control Tux the Linux Penguin as he races down steep, snow-covered mountains grabbing fish. 3D and very well done.

FreeCiv - a multiplayer strategy game.

Classic Gaming - MAME ROMs

Crack Attack - based on the Super Nintendo classic Tetris Attack

Go - game

IceBreaker - Jezzball clone

Maelstrom - networkable, Asteroids clone

Angband - best text based dungeon crawl ever!

Chromium BSU - neat-o OpenGL 2D shooter

Egoboo - is a 3-D top-down dungeon crawler

Liquid War - is addictive, fast-paced, and easy to learn. Perhaps best of all, there's no installation routine. Unzip it and run it, and watch your cat disappear under a pile of laundry as everything outside the game ceases to be interesting.

Multimedia

CDex

OggDS - mediaplayer codec for Ogg Vorbis

Virtual Dub - video capture and linear editing

VLC is a free cross-platform media player. It supports a large number of multimedia formats, without the need for additional codecs. It is available for almost every OS. It needs little CPU power.

Zinf is Not FreeAmp - MP3 and Ogg player

Audacity - Audacity is a free audio editor. You can record sounds, play sounds, import and export WAV, AIFF, and MP3 files, and more. Use it to edit your sounds using Cut, Copy and Paste (with unlimited Undo), mix tracks together, or apply effects to your recordings. It also has a built-in amplitude envelope editor, a customizable spectrogram mode and a frequency analysis window for audio analysis applications. Built-in effects include Bass Boost, Wahwah, and Noise Removal, and it also supports VST plug-in effects.

System Tools

7-Zip - a file archiver with highest compression ratio. Supported formats: 7z, ZIP, CAB, RAR, ARJ, GZIP, BZIP2, TAR, CPIO, RPM and DEB.

JZip - WinZip replacement

VNC - remote control software

LiteStep - a replacement desktop environment released under the GPL.
